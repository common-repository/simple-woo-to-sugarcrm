Automatically send WooCommerce customers to your SugarCRM dashboard as leads.

== Description ==
Automatically send WooCommerce customers to your SugarCRM dashboard as leads after they check out. This is a great way to avoid services like Zapier.

== Installation ==
1. Install and Activate Simple Woo to SugarCRM.
2. Visit "Settings" on the Wordpress Admin Dashboard.
3. Click on "Woo to Sugar Settings".
4. Fill in your SugarCRM Username, Password, Team ID (optional), and Business Type (also optional).
5. Save Changes.
6. Sit back and relax as you watch converted WooCommerce customers populate the leads on your Sugar dashboard.
